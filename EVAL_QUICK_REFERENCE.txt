🎯 EVALUATION DETAILS
================================================================================

For each (model, dataset, task) combination:
  1. Load model checkpoint
  2. Load dataset
  3. Run evaluation on val set
  4. Run evaluation on test set
  5. Save metrics as JSON

Metrics computed:
  • Accuracy
  • F1 Score
  • AUC-ROC
  • Loss (+ optional: MRR, HITS@K)

Output per evaluation:
  {
    "model_path": "...",
    "dataset": "midterm",
    "task": "node_masking",
    "timestamp": "...",
    "val_results": {"accuracy": 0.92, ...},
    "test_results": {"accuracy": 0.89, ...}
  }

🔧 CUSTOMIZATION
================================================================================

Change which models/datasets/tasks to evaluate:
  Edit: scripts/generate_eval_jobs.py
    • MODELS = [...]
    • DATASETS = [...]
    • TASKS = [...]
  Then regenerate: python scripts/generate_eval_jobs.py

Adjust SLURM settings:
  python scripts/generate_eval_jobs.py --array_parallel 8   (less parallel)
  python scripts/generate_eval_jobs.py --mem 64G            (more memory)
  python scripts/generate_eval_jobs.py --time_limit 6:00:00 (longer time)
  python scripts/generate_eval_jobs.py --gpu_type p100      (typed GPU request)

Test single evaluation:
  python scripts/eval_cross_dataset.py \
    --model_path /path/to/model \
    --dataset midterm \
    --task node_masking
